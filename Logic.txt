Старторая страница:
    Если пользователь неавторизован, то форма для входа
    Иначе, страница с контактами

Вход на страницу контактов
    1. Отправить запрос в бд для получения данных по контактам
    2. Пока не пришел ответ, отобразить спиннер
    3. Когда пришел ответ, спиннер и вывести на странице все полученные контакты

Удаление контактам
    При нажатии на кнопку "Удалить"
        1. Открыть всплывающее окно с сообщением "Вы действительно хотите удалить контакт <name> из списка контактов?"
        2.1 При нажатии на кнопку "Нет" закрыть всплывающее окно
        2.2 При нажатии на кнопку "Да" 
            2.2.1 Закрыть всплывающее окно
            2.2.2 Отправить запрос в бд на удаление контакта
            2.2.3 Пока не пришел ответ, отобразить спиннер
            2.2.4 Когда пришел ответ убрать спиннер и вывести сообщение "Контакт успешно удален"
            2.2.5 Выполнить логику "Вход на страницу контактов"

Редактирование контакта
    При нажатии на кнопку "Редактировать"
        1. Перейти на форму с редактированием контакта. Форма состоит из:
            а. Поле "Имя" активное для редактирования
            б. Поле "Телефон" активное для редактирования
            г. Кнопка "Отмена". Активна всегда
            д. Кнопка "Сохранить". Активно только тогда, когда есть изменения
        2.1 При нажатии на кнопку "Отмена" закрыть всплывающее окно. Логика завершается.
        2.2 При нажатии на кнопку "Сохранить"
            2.2.1 Если кнопка неактивна, то ничего не делать
            2.2.2. Если кнопка активна, то открыть всплывающее окно с сообщением "Вы уверены, что хотите сохранить изменение данных?"
                2.2.2.1. При нажатии на "Нет", закрыть всплывающее окно. 
                2.2.2.2. При нажатии на "Да"
                    2.2.2.2.1 Закрыть всплывающее окно
                    2.2.2.2.2 Отправить запрос в бд на редактирование контакта
                    2.2.2.2.3 Пока не пришел ответ, отобразить спиннер
                    2.2.2.2.4 Когда пришел ответ вывести сообщение "Контакт успешно удален"
                    2.2.2.2.5 Выполнить логику "Вход на страницу контактов"
