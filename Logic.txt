Старторая страница:
    Если пользователь неавторизован, то форма для входа
    Иначе, страница с контактами

Вход на страницу контактов
    1. Отправить запрос в бд для получения данных по контактам
    2. Пока не пришел ответ, отобразить спиннер
    3. Когда пришел ответ, спиннер и вывести на странице все полученные контакты

Удаление контактам
    При нажатии на кнопку "Удалить"
        +1. Открыть всплывающее окно с сообщением "Вы действительно хотите удалить контакт <name> из списка контактов?"
        +2.1 При нажатии на кнопку "Нет" закрыть всплывающее окно
        2.2 При нажатии на кнопку "Да" 
            2.2.1 Отправить запрос в бд на удаление контакта
            2.2.2 Пока не пришел ответ, отобразить спиннер
            +2.2.3 Когда пришел ответ убрать спиннер и вывести сообщение "Контакт успешно удален"
            +2.2.4 Закрыть всплывающее окно
            2.2.5 Выполнить логику "Вход на страницу контактов"

Редактирование контакта
    При нажатии на кнопку "Редактировать"
        0. Если была открыта форма Добавления контакта, закрыть ее
        1. Под строкой с контактом открыть форму для редактирования контакта. Форма состоит из:
            а. Поле "Имя" активное для редактирования
            б. Поле "Телефон" активное для редактирования
            г. Кнопка "Отмена". Активна всегда
            д. Кнопка "Сохранить". Активна всегда
        2.1 При нажатии на кнопку "Отмена" убрать форму изменения контакта. Логика завершается.
        2.2 При нажатии на кнопку "Сохранить"
            2.2.1 Если не прошла валидация, то вывести сообщение об ошибке
            2.2.2. Если прошла валидация, то открыть всплывающее окно с сообщением "Вы уверены, что хотите сохранить изменение данных?"
                2.2.2.1. При нажатии на "Отмена", закрыть всплывающее окно. 
                2.2.2.2. При нажатии на "Да"
                    2.2.2.2.1 Отправить запрос в бд на редактирование контакта
                    2.2.2.2.2 Пока не пришел ответ, отобразить спиннер
                    2.2.2.2.3 Когда пришел ответ вывести сообщение "Контакт успешно удален"
                    2.2.2.2.4 Закрыть всплывающее окно
                    2.2.2.2.5 Выполнить логику "Вход на страницу контактов"
